React Project

Pet Adoption App

Developed a Single Page Application using React. The app makes use of public API to load pets data and allows users to search for different pets using a wide variety of filters i-e Location, Animal & Breed. The user can also see different photos of pets and pet details on the details page and when he clicks on the Adopt Me button the app takes the user to the actual petfinder app. All these functionalities are implemented using different React concepts i-e hooks, context api, portals & routing.Link of this project is Pet Adoption App.
